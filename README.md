# dashboard

Dashboard application for Vero Tecnologia using React (create react app) and styled components.

<!-- <img src='./assets/prints/screen.png' style="width:200px;"/> -->

## How to install dependencies?

### On project's folder:

- using yarn

```sh
yarn
```

- using npm

```sh
npm install
```

## How to run?

- yarn

```sh
yarn start
```

- npm

```sh
npm start
```
