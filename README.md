# dashboard

Dashboard application for Vero Tecnologia using React (create react app) with typescript and styled components.

<img src='./public/app.gif' />

## How to install dependencies?

### On project's folder:

- using yarn

```sh
yarn
```

- using npm

```sh
npm install
```

## How to run?

### On project's folder:

- yarn

```sh
yarn start
```

- npm

```sh
npm start
```
